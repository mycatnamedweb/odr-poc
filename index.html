Game PoC - iOS ODR


Example of ODR resource hosted on this server:
https://gabriele-sacchi.github.io/odr-poc/games/AssetPackManifest.plist


Curl:

curl -v 'https://gabriele-sacchi.github.io/odr-poc/games/AssetPackManifest.plist' \ -X GET \ -H 'Origin: iosapp://www.williamhill.com' \ -H 'Accept: */*' \ -H 'User-Agent: Mozilla/5.0 (iPhone; CPU iPhone OS 12_2 like Mac OS X) AppleWebKit/605.1.15 (KHTML, like Gecko) Mobile/15E148'



Fetch API: (send this from an iOS simulator)

fetch('https://gabriele-sacchi.github.io/odr-poc/games/AssetPackManifest.plist') .then(res => { console.debug('Got resp.'); return res.text(); }) .then((res) => console.debug('>> ', res.substring(0, 100)));
